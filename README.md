# Supermercado Família - Sistema de Pedidos Online

Sistema de pedidos online para o Supermercado Família, localizado em Porto Velho/RO.

## 📋 Funcionalidades

- ✅ Sistema de carrinho de compras completo
- ✅ Três etapas de checkout otimizadas
- ✅ Carrinho flutuante com preview dos itens
- ✅ Filtros por categoria e busca de produtos
- ✅ Seleção de zona de entrega com taxas diferenciadas
- ✅ Multiplas formas de pagamento
- ✅ Integração com WhatsApp para finalização do pedido
- ✅ Total responsivo para mobile e desktop
- ✅ Acessibilidade (ARIA, keyboard navigation, screen reader support)

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3 (com variáveis CSS, Flexbox, Grid)
- JavaScript Vanilla (ES6+)
- Font Awesome Icons
- Google Fonts (Poppins, Inter)
- LocalStorage para persistência de dados
